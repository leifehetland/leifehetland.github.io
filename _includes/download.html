Projects

History of You

History of You is a geo-locational storytelling telling application where a user is able to select a location utilizing the Google Maps API, and then write a story for that location. By doing so, a user can see other user's narratives. AngularJS front end with Firebase database.

Conway's Game of Life

A unit tested, C# implementation of Conway's cellular automaton zero-player game.

Trick Or Boom!

A game where the player controls a pumpkin guy named Jacques through a series of graveyard mazes fending off slime monsters and other spooky things. Jacques drops bombs to blast his way through these enemies and environments. Phaser/JS game with an MVC shell and database.

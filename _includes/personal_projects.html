Personal Projects

Is Anyone Out There?

Global Game Jam 2018- A prototype for a first person narrative experience using C# and the Unity game engine based on the theme of transmission. The player is a woman who is receiving mysteries signals that generate memories that the player explores.

Umi-chan vs Cephalotron

Global Game Jam 2017- A protoype for a rhythm/action game using C# and the Unity game engine based on the theme of waves. Worked with a team of artists, a musician, and developers.

Ritual Reversal

Global Game Jam 2016- A protoype for a platformer game using C# and the Unity game engine based on the theme of ritual. Worked with a team of artists, musicians, and developers.

Conway's Game of Life

A unit tested, C# implementation of Conway's cellular automaton zero-player game.

Trick Or Boom!

A game where the player controls a pumpkin guy named Jacques through a series of graveyard mazes fending off slime monsters and other spooky things. Jacques drops bombs to blast his way through these enemies and environments. Built using PhaserJS.

History of You

History of You is a geo-locational storytelling application where a user is able to select a location utilizing the Google Maps API, and then write a story for that location. By doing so, a user can see other user's narratives. AngularJS front end with Firebase database.
